    What does the throw keyword do?

  • The throw keyword throws a user defined exception to be caught with a try catch statement.

    • What is the difference between a TypeError and a ReferenceError?

  • A type error is an error that occures when a variable contains a value, but the operation trying to be performed on the value is not valid for the type. A reference error occurs when a variable is called that doesn't exist at all.
